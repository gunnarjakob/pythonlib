Installation

Clone the package from https://github.com/gunnarvoet/gvpy. Then install gvpy by changing into the root directory and running

>>> python setup.py install

or using pip

>>> pip install .

To install in developer mode run either

>>> python setup.py develop

or

>>> pip install -e .

Now in python you should be able to run

import gvpy

which will provide several sub-modules as documented here.
