============
Installation
============

To install in regular mode:

.. code-block:: shell
    
    $ python setup.py install

To install in developer mode

.. code-block:: shell

    $ python setup.py develop

or using pip:

.. code-block:: shell

    $ pip install -e .

To import run:

.. code-block:: python

    import gvpy as gv

This will provide several submodules under `gv`
